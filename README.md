###LogLens AI

AI-powered log monitoring and analysis system.

~Overview~

LogLens collects application logs, analyzes them using AI, and provides:

-Error summaries
-Severity classification
-Suggested fixes
-Alerts for critical issues

##Tech Stack
-Node.js
-TypeScript
-Express
-PostgreSQL (planned)
-Redis + BullMQ (planned)
-OpenAI API (planned)
